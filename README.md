# TS3 Scripting Template

This .NET project is designed to get you started in creating scripting mods for The Sims 3. This includes a basic hello world-like example from [Sims Wiki](http://simswiki.info/wiki.php?title=Tutorial:Sims_3_Pure_Scripting_Modding).

## Requirements

### Prerequisites

* Visual Studio
  * .NET desktop workload
* TS3: Patch 1.69 (Origin)
